## üé∂ Spectral Sound Processing Assignments

This repository contains the **9 assignments** for the course Digital Signal Processing for Sound and Music focused on **spectral processing techniques** of relevance for the description and transformation of sounds. The goal is to develop the basic theoretical and practical knowledge needed to **analyze, synthesize, transform, and describe audio signals** in the context of music applications.

### üìù Assignments Overview

| \# | Notebook | Topic | Key Concepts |
| :---: | :--- | :--- | :--- |
| **E1** | `E1-Python-and-sounds.ipynb` | **Python and Sound I/O** | Basic Python, audio file handling, signal plotting. |
| **E2** | `E2-Sinusoids-and-DFT.ipynb` | **Sinusoids and DFT** | Understanding basic sinusoidal components and implementing the **Discrete Fourier Transform (DFT)**. |
| **E3** | `E3-Fourier-properties.ipynb` | **Fourier Properties** | Exploring the mathematical properties of the Fourier Transform (linearity, time/frequency shift, etc.). |
| **E4** | `E4-STFT.ipynb` | **Short-Time Fourier Transform (STFT)** | Implementing the fundamental technique for time-varying spectral analysis (**Spectrograms**). |
| **E5** | `E5-Sinusoidal-model.ipynb` | **Sinusoidal Model** | Analysis and synthesis of sound using tracked spectral peaks. |
| **E6** | `E6-Harmonic-model.ipynb` | **Harmonic Model** | A specialized sinusoidal model focusing on **pitched sounds** and their **harmonics**. |
| **E7** | `E7-Sinusoidal-plus-residual-model.ipynb` | **Sinusoidal+Residual Model** | Advanced analysis model separating sound into pitched/tonal (sinusoidal) and noisy/transient (residual) components. |
| **E8** | `E8-Sound-transformations.ipynb` | **Sound Transformations** | Applying analysis-synthesis models to perform audio effects like **time-stretching** and **pitch-shifting**. |
| **E9** | `E9: Sound and music description` | **Sound and Music Description** | Extracting and analyzing higher-level features for **Music Information Retrieval (MIR)** (e.g., rhythm, tempo, timbre). |
