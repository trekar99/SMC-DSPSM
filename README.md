This course focuses on the spectral processing techniques of relevance for the description and transformation of sounds, developing the basic theoretical and practical knowledge with which to analyze, synthesize, transform and describe audio signals in the context of music applications.

The course is based on open software and content. The demonstrations and programming exercises are done using Python, and the references and materials for the course come from open online repositories. We are also distributing with open licenses the software and materials developed for the course.

All the course materials are in: https://github.com/MTG/sms-tools-materials. All the assignments are in: https://github.com/MTG/sms-tools-materials/tree/main/exercises
